# Support

## Getting Help

If you need help using OpenClaw HUD, please open a GitHub issue.

- Bug report: use the **Bug Report** issue template
- Feature request: use the **Feature Request** issue template
- Security issue: follow [SECURITY.md](./SECURITY.md) and do not file public details

## What to Include

To help us resolve issues quickly, include:

- OS and version
- Node.js version (`node -v`)
- OpenClaw HUD version/commit
- OpenClaw version (if known)
- Repro steps
- Expected behavior
- Actual behavior
- Relevant logs or screenshots (redacted)

## Response Expectations

- Initial triage target: within 3 business days
- Follow-up cadence: at least weekly for active issues

## Scope

Support is best-effort and focused on:

- Reproducible bugs in this repository
- Setup and local runtime issues
- Regression reports

We may close issues that are out of scope, missing repro details, or are
duplicates of existing issues.
